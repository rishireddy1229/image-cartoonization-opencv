# Image Cartoonization using OpenCV and Streamlit

This project implements an image cartoonization pipeline using classical computer vision techniques. It provides both a command-line interface and an interactive web application for converting real-world images into stylized cartoon outputs.

---

## Overview

The system processes an input image through a sequence of steps including smoothing, edge detection, and color quantization. The final cartoon image is generated by combining edge masks with simplified color regions.

Two methods are supported for color quantization:
- K-Means Clustering
- Mean Shift Filtering

---

## Features

- Interactive Streamlit web application
- Command-line interface support
- Image upload and real-time processing
- Cartoonization using clustering-based techniques
- Edge detection using adaptive thresholding
- Edge-preserving smoothing using bilateral filtering
- Evaluation using SSIM and MSE
- Downloadable output images
- Automatic saving of results to `output_images/`

---

## Methodology

The cartoonization pipeline consists of:

1. Bilateral filtering for smoothing while preserving edges
2. Adaptive thresholding for edge detection
3. Color quantization using:
   - K-Means clustering
   - Mean Shift filtering
4. Bitwise combination of edges and color layers
