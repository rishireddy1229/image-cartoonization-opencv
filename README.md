# Image Cartoonization using OpenCV and Streamlit

This project implements an image cartoonization pipeline using classical computer vision techniques. It provides both a command-line interface and an interactive web application for converting real-world images into stylized cartoon outputs.

---

## Overview

The system processes an input image through a sequence of image processing steps including smoothing, edge detection, and color quantization. The final output is generated by combining edge masks with simplified color regions.

Two approaches are supported for color quantization:
- K-Means Clustering
- Mean Shift Filtering

---

## Features

- Interactive web interface for image upload and visualization
- Command-line interface for batch processing
- Edge-preserving smoothing using bilateral filtering
- Edge detection using adaptive thresholding
- Color quantization using clustering techniques
- Downloadable output images
- Automatic saving of results to disk

---

## Methodology

The cartoonization pipeline consists of the following steps:

1. Apply bilateral filtering to smooth the image while preserving edges
2. Detect edges using adaptive thresholding
3. Perform color quantization:
   - K-Means clustering (fixed number of clusters)
   - Mean Shift filtering (mode-seeking segmentation)
4. Combine edge mask with quantized image using bitwise operations

## Installation

Install dependencies:


pip install -r requirements.txt


---

## Running the Application

### Web Interface


python -m streamlit run app.py


Open the application in your browser at:
http://localhost:8501

---

### Command-Line Usage


python main.py --input path_to_image --method kmeans


Options:
- `--method kmeans`
- `--method meanshift`

---

## Output

Processed images are saved in the `output_images/` directory. The web interface also provides an option to download the generated image.

---

## Evaluation

The project includes basic evaluation metrics:
- Structural Similarity Index (SSIM)
- Mean Squared Error (MSE)

---

## Notes

This project uses classical computer vision techniques and does not rely on deep learning models. It is intended for understanding image stylization using traditional approaches.
